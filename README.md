# Telegram Giveaway Bot (Бот для Розыгрышей)

Простой Telegram-бот для проведения розыгрышей в чатах.

## Возможности

*   Создание нового розыгрыша с указанием приза.
*   Участие пользователей в активном розыгрыше.
*   Определение случайного победителя.

## Конфигурация

### 1. Получение Telegram Bot Token

*   Откройте Telegram и найдите бота `@BotFather`.
*   Отправьте ему команду `/newbot`.
*   Следуйте инструкциям, чтобы задать имя и username для вашего бота. Username должен заканчиваться на `bot` (например, `MyGiveawayTestBot`).
*   После успешного создания BotFather пришлет вам токен. **Это секретный ключ, не делитесь им ни с кем!**

### 2. Настройка токена для бота

У вас есть два способа предоставить токен боту:

*   **Рекомендуемый способ: Переменная окружения**
    Установите переменную окружения `TELEGRAM_TOKEN` равной вашему токену.
    *   Для Linux/macOS (временно для текущей сессии терминала):
        ```bash
        export TELEGRAM_TOKEN="ВАШ_СЕКРЕТНЫЙ_ТОКЕН"
        ```
        Для постоянной установки добавьте эту строку в ваш конфигурационный файл оболочки (например, `~/.bashrc` или `~/.zshrc`), затем выполните `source ~/.bashrc` (или соответствующий файл).
    *   Для Windows (временно для текущей сессии PowerShell):
        ```powershell
        $env:TELEGRAM_TOKEN="ВАШ_СЕКРЕТНЫЙ_ТОКЕН"
        ```
        Для постоянной установки используйте "Панель управления" -> "Система и безопасность" -> "Система" -> "Дополнительные параметры системы" -> "Переменные среды".

*   **Альтернативный способ: Ввод при запуске**
    Если переменная окружения `TELEGRAM_TOKEN` не установлена, бот (`giveaway_bot/bot.py`) запросит токен при запуске в консоли. Просто вставьте ваш токен, когда появится соответствующий запрос.

## Запуск бота

После установки зависимостей и настройки токена, запустите бота командой:

```bash
python giveaway_bot/bot.py
```

Вы должны увидеть в консоли сообщение о том, что токен получен (будут показаны первые 6 символов) и бот запускается.

## Доступные команды

*   `/start` - Приветственное сообщение от бота.
*   `/new_giveaway <название приза>` - Начать новый розыгрыш в текущем чате. Например: `/new_giveaway Билет в кино`
*   `/enter` - Присоединиться к текущему активному розыгрышу в чате.
*   `/draw_winner` - Определить победителя текущего розыгрыша и завершить его. (На данный момент может быть вызван любым пользователем в чате, где идет розыгрыш).
