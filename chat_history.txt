## Команды для запуска

### Запуск API сервера
```bash
python app.py
```

### Запуск в автоматическом режиме
```bash
python app.py --auto
```

### С настройкой интервала проверки (в секундах)
```bash
python app.py --auto --interval 60
```

### С фильтрацией по ключевым словам
```bash
python app.py --auto --keywords "python,автоматизация"
```
